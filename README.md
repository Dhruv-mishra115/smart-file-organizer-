\# Smart File Organizer (Python)



A Python automation tool that organizes messy folders by automatically sorting files into categorized directories.



\## Features

\- Organizes files based on extension

\- Recursive folder scanning

\- Automatic folder creation

\- Duplicate file protection

\- Command Line Interface support



\## Technologies Used

\- Python

\- os module

\- shutil module

\- Command Line Interface (CLI)



\## How to Run



Clone the repository:



git clone https://github.com/Dhruv-mishra115/smart-file-organizer-.git



Navigate into the project folder:



cd smart-file-organizer-



Run the program:



python Organizer.py test\_folder



\## Example



Before running:



test\_folder/

&nbsp;   photo.jpg

&nbsp;   video.mp4

&nbsp;   notes.txt



After running:



test\_folder/

&nbsp;   Images/

&nbsp;       photo.jpg

&nbsp;   Videos/

&nbsp;       video.mp4

&nbsp;   Documents/

&nbsp;       notes.txt
